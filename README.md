# Link-Lite

리더십 진단 및 팀 궁합 분석 시스템

## 프로젝트 개요

Link-Lite는 사용자의 리더십 유형을 진단하고, 팀원들의 팔로워십 유형과의 궁합도를 분석하는 웹 애플리케이션입니다.

## 주요 기능

### 리더십 진단
- **23개 문항**: 체계적인 리더십 평가
- **6점 척도**: 매우그렇지않다(1) ~ 매우그렇다(6)
- **3가지 상위요인**: 공유및참여, 상호작용, 성장촉진
- **12개 하위요인**: 세부적인 리더십 역량 분석
- **8가지 리더십 유형**: 조합에 따른 상세한 분류

### 팀 궁합 분석
- **5가지 팔로워십 유형**: Driver, Doer, Supporter, Thinker, Follower
- **궁합도 점수**: 0-100점 척도의 상세한 분석
- **맞춤형 조언**: 리더십 유형별 개선 방안 제시

## 리더십 유형 분류

각 차원의 평균 점수가 4.5 이상이면 'H(High)', 미만이면 'L(Low)'로 분류:

- **HHH - 참여코칭형**: 팀 성장에 대한 관심과 코칭으로 구성원의 의견을 수용하고 자율적 실행을 촉진하는 스타일
- **HHL - 참여친밀형**: 팀 소통/정서 케어를 바탕으로 협업의 연결을 강화하는 스타일
- **HLH - 참여비전형**: 큰 그림을 제시하고 자율 참여를 끌어 성과를 만드는 스타일
- **HLL - 참여실무형**: 역할 배분, 일정, 진행을 체계적으로 관리해 운영을 안정화하는 스타일
- **LHH - 개별코칭형**: 1:1 맞춤형 코칭을 통해 개인의 업무적 성장과 비전을 촉진하는 스타일
- **LHL - 개별친밀형**: 1:1 신뢰, 섬세한 지원으로 몰입과 관계 안정을 높이는 스타일
- **LLH - 개별비전형**: 팀 개인의 미래 비전과 성장을 1:1 케어를 통해 새로운 관점에서 방향을 제시하는 스타일
- **LLL - 과도기형**: 리더십 활동의 위축기 또는 리더십 정체성을 재정립하는 과도기

## 하위요인 구조

### 공유및참여 (6문항)
- s1_진행공유: 팀원 간 업무 진행 상황 공유
- s2_목표공유: 팀 목표와 방향성 공유
- s3_의견반영: 팀원 의견 적극 수용

### 상호작용 (8문항)
- i1_정서케어: 팀원의 정서적 지원
- i2_문제해결: 업무 문제 해결 지원
- i3_개방성: 솔직한 소통 환경 조성
- i4_팀역동이해: 팀원 간 이해 촉진

### 성장촉진 (9문항)
- g1_실패학습: 실수로부터의 학습
- g2_도전촉진: 새로운 시도 독려
- g3_결과수준: 결과물 품질 기준 제시
- g4_주도성촉진: 자율적 소통 촉진
- g5_일의의미: 업무의 의미와 성장 연결

## 사용 방법

### 설치 및 실행

```bash
# 프로젝트 클론
git clone <repository-url>
cd Link-Lite

# 개발 서버 실행
python3 -m http.server 8080 --directory public
```

브라우저에서 `http://localhost:8080` 접속

### 진단 과정

1. **리더십 진단**: 23개 문항에 6점 척도로 응답
2. **결과 확인**: 3가지 차원 점수와 리더십 유형 확인
3. **팀원 선택**: 팀원들의 팔로워십 유형 선택 (복수 선택 가능)
4. **궁합 분석**: 각 팀원과의 궁합도 및 개선 방안 확인

## 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Design**: 반응형 웹 디자인, 모던 UI/UX
- **시각화**: SVG 기반 레이더 차트
- **배포**: Netlify (Branch deploys 지원)

## 특징

- 반응형 디자인: 모바일, 태블릿, 데스크톱 지원
- 모던 UI: 직관적이고 아름다운 사용자 인터페이스
